# 🎬 《十字门代码》视频动画与图文全景方案（开发版）

## 🎬 5个核心故事动画设置（每个15秒）

### 1. 【第一关：密文影印本】—— 序幕：白银消失与银币交托
* **触发时机**：学生在地面层(G/F)闸机口/咨询台扫描第一个二维码后。
* **动画内容（15s）**：
    * **画面**：1634年黑夜暴风雨大作，明代旗舰“海神号”在十字门触礁沉没（5s）。镜头穿过层层泥沙，展现银箱受到近海底回流吸引没入泥沙的奇景（5s）。画面瞬间拉回现代，老馆员那只沧桑的手将一枚**【被氧化发黑的葡萄牙银币】**轻轻推向学生（玩家视角），手机屏幕定格在《崇祯七年·闽海贸易账目折子》影印页上（5s）。
* **作用**：抛出核心悬念，交代历史背景，并用动画形式**亲手将核心初始道具“葡萄牙银币”交到学生手中**。

### 2. 【第四关：海底推移假说】—— 追寻：航海家的疯狂推论
* **触发时机**：学生在地面层(G/F)C区参考咨询台扫码成功后。
* **动画内容（15s）**：
    * **画面**：1982年昏暗的澳门档案室，老年的安东尼奥颤抖着手指，划过抄本上的“双十字”（5s）。随后切入他站在船头观测海流、在桌前手绘推移路线图的剪影（5s）。最后画面定格在他将厚厚的笔记本打包，寄往横琴的无名邮戳上，屏幕打出他的手写字幕：“答案或许就在书架之间”（5s）。
* **作用**：引入关键人物安东尼奥，解锁道具【推移假说手稿】，将玩家视线从“古代海洋”拉向“现代图书馆”。

### 3. 【第八关：图书馆地桩】—— 碰撞：历史与现实的重合
* **触发时机**：学生上到二楼(2/F)B区多媒体服务台扫码确认位置后。
* **动画内容（15s）**：
    * **画面**：2011年尘土飞扬的横琴工地，巨大的打桩机轰鸣着砸下第一根地桩（5s）。镜头快速切入地下5米，伴随着一声令人牙酸的沉闷巨响，泥浆中混着贝壳和咸水味喷涌而出（5s）。镜头一转，办公室里的年轻馆员猛然抬头，手中正拿着安东尼奥的笔记手稿，眼中满是震惊（5s）。
* **作用**：全剧最大的剧情转折点，解锁【地桩施工记录】。用声音和画面的物理冲击，向学生证明“地下真的有东西”。

### 4. 【第十一关：解密钥匙】—— 觉醒：全道具时空大对齐
* **触发时机**：学生在讨论室门外屏幕扫码并成功模拟预约后。
* **动画内容（15s）**：
    * **画面**：镜头以极快的速度闪回前10关学生在手机里收集到的**所有道具碎片**（影印页、旧海图、施工记录、木梁碎片等）（4s）。随后，这些道具在屏幕上自动分为“古代组”与“现代组”，像精密齿轮一样在半空中自动咬合、旋转、重叠对齐（5s）。最终，它们化为一张散发着微光的“图书馆终极路线图”，终点精准指向一楼核心特藏区（6s）。
* **作用**：解谜的超级爽点爆发！将所有获得的道具在动画中**一键拼合**，为最后一关冲刺蓄力。

### 5. 【第十二关：继承者之证】—— 尾声：传承与终极悬念
* **触发时机**：学生在一步(1/F)古籍馆/特藏展柜扫码完成终极答题的瞬间。
* **动画内容（15s）**：
    * **画面**：深夜寂静的图书馆，学生（玩家视角）从背包中拿出**【从第一关带到现在的葡萄牙银币】**，卡入最后的古籍柜机关，齿轮发出沉闷的转动声（5s）。一扇隐蔽的书架缓缓移开，一缕月光照了进去……（5s）。画面并不拍里面到底有没有白银，而是切向走廊深处，老馆员在阴影里露出了欣慰的微笑，画面渐渐淡出（5s）。
* **作用**：大结局。首尾呼应，完美利用初始道具银币开启机关，留白致敬，手机端同时弹出终极道具【双十字银印】。
