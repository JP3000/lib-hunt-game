# 鄭芝龍的寶藏 II：AI 媒體素材製作方案

> 使用 liblibAI 智能圖片 v2 模型生成道具圖片與影片素材。
> 工作流程：**先出圖 → 圖作為參考幀 → 生成影片**。

---

## 一、整體視覺風格定義

| 維度 | 方向 |
|------|------|
| **時代背景** | 明末崇禎年間（1634）+ 葡萄牙大航海時代 + 現代澳門大學圖書館 |
| **色調** | 暖黃做舊基底（ parchment / sepia ），局部點綴暗金、深藍（海事元素） |
| **光影** | 仿自然光——圖書館內用窗戶側光 + 書架間柔和陰影；歷史物品用頂光突出紋理 |
| **質感關鍵詞** | aged paper texture, worn edges, burnt edges, wax seal, nautical chart, silver tarnish, ink bleed, calligraphy brush strokes, vellum |
| **統一後處理** | subtle vignette, slight desaturation, film grain, warm tone |

### Level 8-12 道具：五常卡牌套裝（仁、義、禮、知、信）

> ⚠️ **重要**：這五張卡牌必須統一設計，確保風格一致，且能拼合成五角形。

**統一設計規範：**

| 屬性 | 規格 |
|------|------|
| 尺寸比例 | 3:4（縱向卡片） |
| 正面設計 | 居中大字（毛筆楷書）+ 左上角拉丁轉寫 + 對應圖騰紋樣 |
| 背面設計 | 居中拉丁文釋義 + 下方小字線索文字 + 統一的航海繩索邊框 |
| 材質 | 啞光米黃卡紙，邊緣燙暗金，輕微磨損做舊 |
| 拼合時 | 五張背面朝上，按仁→義→禮→知→信順時針排列，形成完整五角形 |

**五張卡牌的個別內容：**

| 9 | 義 | gni — gratitudo | 天平與羽毛筆交叉 | 代表對知識傳播的使命感與正義，保護古籍、尊重知識 |
