## 🧭 地面层 (G/F) 入馆与基础服务篇

### 🧩 第一关：密文影印本
* **📚 图书馆任务（1）**：**入馆注意事项**（认识入馆闸机、携带证件、不可带饮料食物等规定）。
* **📍 现实扫码目标**：**地面层 (G/F) B区 —— 入馆闸机口旁 / 资讯服务台台面**
* **🎬 故事动画（15s）【序幕：白银消失与密文诞生】**
    * *新生在闸机口或服务台前扫描第一个二维码确认位置后触发。1634年黑夜，海神号在十字门巨浪中触礁沉没，白银没入泥沙。陈逢春在账目折子上写下密文和“双十字”符号。画面拉回现实，定格在玩家手机里的《崇祯七年·闽海贸易账目折子》影印本上。*
* **🔍 解谜设计**：手机弹出影印本上的“入馆隐写密文”，要求学生对照眼前服务台张贴的《入馆须知》。例如：“提取须知中第3、7、21个字”，在手机上拼出暗号，解锁下一关的地点提示。

### 🧩 第二关：双十字印记
* **📚 图书馆任务（2）**：**楼层分布**（熟悉G/F到6/F的整体布局，学会看楼层索引图）。
* **📍 现实扫码目标**：**地面层 (G/F) B区 —— 资讯共享空间（大厅大屏幕或整体楼层分布图旁）**
* **📸 图文说明**：
    * **【图】**：影印本背面放大褪色的“1982·安东尼奥”字迹，旁边有用红圈勾勒的“双十字”。
    * **【文】**：老馆员缓缓推过一枚发黑的银币：“安东尼奥的笔记被我拆散了。想要找到它们，你得先看懂这座建筑的脉络。去看看大厅的楼层分布，那里藏着他留下的第一个坐标。”
* **🔍 解谜设计**：学生物色到大厅楼层分布图并成功扫码后，手机要求指出“特藏阅览室”在哪一层哪个区域（正确答案：1/F A区）。答对后，手机给出下一关的指引。

### 🧩 第三关：海神号传闻
* **📚 图书馆任务（3）**：**馆藏目录和发现系统**（学习使用图书馆检索机OPAC终端检索图书）。
* **📍 现实扫码目标**：**地面层 (G/F) B区 —— 新书区旁 / 馆藏检索机(OPAC)终端屏幕旁**
* **📸 图文说明**：
    * **【图】**：一张泛黄的澳门地方志古籍残页，上面有毛笔字记录的“海眼”、“吞金”等字样。
    * **【文】**：“船沉了，但没有一两白银浮上来。水手说是海眼吞金。但账房陈逢春知道，它只是去了凡人看不见的地方。用这里的‘发现系统’，把那些被时间掩埋的名字搜出来。”
* **🔍 解谜设计**：学生找到检索机并扫描机身侧面的二维码。手机联动要求学生在检索机上实际输入关键词“十字门”或“郑芝龙”，并将搜索到的某本特定书籍的**索书号**输入手机，即可过关。

## 🌊 一楼至二楼 (1/F - 2/F) 馆藏与地理推导篇

### 🧩 第五关：十字门旧图
* **📚 图书馆任务（5）**：**怎么查找图书**（利用好书推荐、特藏书库的定位）。
* **📍 现实扫码目标**：**一楼 (1/F) A区 —— 特藏阅览室大门旁 / 澳门坊外墙展示区**
* **📸 图文说明**：
    * **【图】**：1630年代明代与葡萄牙人绘制的十字门水域古地图（大、小横琴岛相夹）。
    * **【文】**：“形如十字，这里曾是吞吐万国商船的白银水道。安东尼奥在图上画下的那条红线，穿越了三百年，正指向你脚下的这片土地。”
* **🔍 解谜设计**：学生来到一楼特藏阅览室门口扫码。手机显示古地图，要求学生观察现场展框里的澳门旧档案，找出古地图上的“大横琴岛”在现代对应什么地方，选择正确答案通关。

### 🧩 第六关：潮汐回流
* **📚 图书馆任务（6）**：**图书怎么排序**（根据索书号在书架上实地找书的规则，排架规律）。
* **📍 现实扫码目标**：**一楼 (1/F) C区 —— 总类书籍区 / 丛书区特定书架侧面**
* **📸 图文说明**：
    * **【图】**：现代海洋研究所的十字门海底暗流模拟图，带有密密麻麻的蓝色流线矢量箭头。
    * **【文】**：“老渔民叫它‘海底推浪’。一种近海底的缓慢回流，像一把巨大的、隐形的手，几百年来，在沙泥的掩护下，将沉重的银箱缓缓推向东北方……它在书架的排列组合中留下了轨迹。”
* **🔍 解谜设计**：手机给出提示：“顺着暗流的方向，寻找代号为 `DS796` 的洋流货架。”学生需要在一楼C区书架间穿行，根据排架规律找到该书架，并扫描贴在书架侧面的二维码进行解谜。

### 🧩 第七关：围海新岸
* **📚 图书馆任务（7）**：**期刊和下载论文**（了解过期期刊区、电子期刊的查阅与下载方法）。
* **📍 现实扫码目标**：**一楼 (1/F) C区 —— 澳门过期期刊区大门或查阅终端旁**
* **📸 图文说明**：
    * **【图】**：一张横琴岛围海造田前后的对比图（左边是茫茫泥滩，右边是现代校区的轮廓）。
    * **【文】**：“沧海桑田。当年的海底斜坡，如今已成寸土寸金的陆地。暗流停下的地方，几百年后的地表上，建起了一座承载知识的建筑。去翻阅那些尘封的期刊，寻找地基动工的记录。”
* **🔍 解谜设计**：学生进入期刊区扫码。手机上会提示一篇特定期刊文章，要求学生在现场的期刊检索终端（或手机连接校网）查出该期刊的“出版年份”或“横琴新区动工的具体年份”，输入数字过关。

### 🧩 第八关：图书馆地桩
* **📚 图书馆任务（8）**：**什么是数据库**（了解LeapSpace、如何使用澳门大学数据库资源检索电子文献）。
* **📍 现实扫码目标**：**二楼 (2/F) B区 —— 多媒体区入口 / 多媒体服务台**
* **🎬 故事动画（15s）【碰撞：历史与现实的重合】**
    * *学生上到二楼，在多媒体服务台前扫码确认位置后触发。2011年横琴工地，巨大的打桩机轰鸣砸下。镜头切入地下5米，伴随着巨响，混着贝壳的泥浆喷涌而出。画面切回当年，办公室里的年轻馆员震惊地看着刚刚收到的安东尼奥手稿。*
* **🔍 解谜设计**：手机上弹出一个“虚拟电脑屏幕”，指导学生认识校内数据库。学生需要回答：“当遭遇AI网络资料真假难辨时，应该使用学校的什么资源来查找最权威的论文？”（正确选项：学网数据库 / LeapSpace），解锁下一步。
